# 🎨 涂鸦画板 (Doodle Page)

一个基于 React + TypeScript 构建的精美在线涂鸦画板工具。

**在线访问**: https://moxccc.github.io/claw-dai

## ✨ 功能特点

- **多彩画笔** - 15 种预设颜色供你选择
- **可调笔刷大小** - 7 种不同粗细的笔刷
- **橡皮擦工具** - 快速擦除绘制内容
- **撤销功能** - 撤销最近的一笔
- **清空画布** - 一键清除所有内容
- **响应式设计** - 支持桌面和移动设备
- **触控支持** - 完美支持触屏设备

## 🛠️ 技术栈

- **React 18** - UI 框架
- **TypeScript** - 类型安全
- **Vite** - 快速构建工具
- **GitHub Pages** - 免费托管部署

## 🚀 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```
